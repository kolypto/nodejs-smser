SMSframework
============

SMS framework with pluggable providers.

Key features:

* Send messages
* Receive messages
* Delivery confirmations
* Handle multiple providers with a single gateway
* Event-oriented
* Promise-based: using the [q](https://npmjs.org/package/q) package
* Unit-tested
